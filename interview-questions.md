# Linux Troubleshooting Interview Questions

## What will you do if a server disk becomes full?

I will check disk usage using df -h, identify large files and directories using du and find, review logs, remove unnecessary files if approved, and request additional storage if needed.

## What will you do if CPU usage is high?

I will use top and ps commands to identify the process consuming CPU, investigate logs, and coordinate with relevant teams.

## What will you do if memory usage is high?

I will check memory usage using free -m and top, identify memory-consuming processes, and investigate application behavior.

## What will you do if a service is down?

I will check service status using systemctl, review logs, verify resources, and restart the service if approved.

## What will you do if SSH is not working?

I will verify network connectivity, check SSH service status, review firewall rules, and examine SSH logs.

## What will you do if an application is not accessible?

I will verify server availability, check services, review logs, verify network connectivity, and coordinate with the application team.
